# SQL Project: Data Analysis for Uber Eats - A Food Delivery Company

## Overview
This project demonstrates my SQL problem-solving skills through the analysis of data for Uber Eats, a leading food delivery company. The project involves setting up the database, importing data, handling null values, and solving a variety of business problems using complex SQL queries.

![Alt Text](https://github.com/RuthMary4/uber-eats-sql-project/blob/main/uber-eats-ERD.png)

## Project Structure

- **Database Setup:** Creation of the `uber_eats_db` database and the required tables.
- **Data Import:** Inserting sample data into the tables.
- **Data Cleaning:** Handling null values and ensuring data integrity.
- **Business Problems:** Solving 20 specific business problems using SQL queries.
